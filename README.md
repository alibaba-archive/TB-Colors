## TB-Colors: Simple can be powerful
A minimalist color palette for use in [TB-UI](https://www.github.com/teambition/tb-ui).

![TB-Colors Logo](./images/tb-colors-screenshot.png)

### Install
```
npm i tb-colors --save
```

### Enjoy Coloring

#### With Stylus
```
@import 'dist/colors'

.tb-logo
  color: $blue

```
